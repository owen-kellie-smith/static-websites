# static-websites

Various static websites rendered on github pages which are approximate copies of websites built with some framework.

https://owen-kellie-smith.github.io/static-websites/


---

## 🚀 GitHub Pages setup (first time only)

1. Push this folder to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to **main branch / root folder**
4. Site goes live at `https://<username>.github.io/<repo>/` ie in this case at `https://owen-kellie-smith.github.io/static-static/`
